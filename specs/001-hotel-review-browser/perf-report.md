# 性能验证报告：花园酒店评论浏览网页

## 验证目标
- 筛选/搜索响应时间：95% 场景 <= 2 秒
- 首屏可浏览内容时间：95% 场景 <= 3 秒

## 测试环境
- 机器：本地开发环境（macOS）
- 数据集：public/enriched_comments.csv（2542 行）
- 方式：基于真实 CSV 数据执行解析与筛选/关键词组合查询基准

## 测试方法
1. 首屏基准：测量 CSV 全量加载与解析耗时（模拟首次进入数据可浏览前准备阶段）。
2. 查询基准：执行 200 次“评分+关键词+日期范围”组合筛选并排序分页，统计均值与 p95。

## 结果
- 数据行数：2542
- 首次加载耗时：13.47 ms
- 查询平均耗时：0.43 ms
- 查询 p95 耗时：0.48 ms

## 结论
- 查询性能显著优于 2 秒目标。
- 首次加载性能显著优于 3 秒目标。
- 当前数据规模下满足计划中的性能预算要求。

## 说明
- 本报告为开发阶段基准，生产环境仍需在真实网络与实际 Insforge 部署条件下复测。
